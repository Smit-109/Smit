Key Concepts

Architecture

Self-Attention

Attention(Q,K,V) = softmax(QKT / √dk)V

Allows tokens to attend to all other tokens regardless of distance.

Training

Next Token Prediction

Self-supervised learning: predict the next token given previous tokens.

L = -Σ log P(wt+1 | w1,...,wt)
Alignment

RLHF

Reinforcement Learning from Human Feedback to align model outputs with human preferences.
